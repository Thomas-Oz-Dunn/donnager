# donnager.rs

Until the rains fall on Olympus Mons.

A library of functions for astrodynamic calculations. 

## Installation
```
cargo install donnager
```

or, alternatively, in your project's `cargo.toml` file:

```
[dependencies]
donnager = "0.1.0"
```

## Modules
- `cosmos`
- `dynamics`
- `propulsion`

## Usage

```
use donnager::*;
```
